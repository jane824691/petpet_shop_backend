# PetPet Shop - Backend Project

Welcome to the PetPet Shop e-commerce backend server. It handles API requests, database interactions, and business logic.

---

## Key Technologies

This project is built with Node.js and relies on the following core technologies:

*   **Runtime Environment:** Node.js v20.19.1
*   **Framework:** Express.js v4.18.2
*   **Database:** MySQL (connected via `mysql2` v3.6.3)
*   **ORM:** Sequelize v6.35.2 for data modeling and queries.
*   **Authentication:** Using `jsonwebtoken` (JWT) for API security and `bcryptjs` for password hashing.
*   **Templating:** EJS v3.1.9 for server-side rendering of views.
*   **File Uploads:** Handled by `multer` v1.4.5-lts.1.
*   **Internationalization (i18n):** `i18next` is used to support multiple languages.
*   **Development:** `nodemon` for automatic server restarts.
*   **Bundling:** `esbuild` for creating a production build.

# PetPet 寵物電商後端

歡迎來到 PetPet Shop 的後端專案！負責處理 API 請求、資料庫互動及核心業務邏輯。

---

## 主要技術

本專案以 Node.js 為基礎，並依賴以下核心技術：

*   **執行環境:** Node.js v20.19.1
*   **網站框架:** Express.js v4.18.2
*   **資料庫:** MySQL (使用 `mysql2` v3.6.3 連接)
*   **ORM:** Sequelize v6.35.2 用於資料模型與查詢
*   **使用者驗證:** 使用 `jsonwebtoken` (JWT) 保護 API 安全，並用 `bcryptjs` 進行密碼雜湊
*   **樣板引擎:** EJS v3.1.9 用於伺服器端視圖渲染
*   **檔案上傳:** 使用 `multer` v1.4.5-lts.1 處理
*   **國際化 (i18n):** 使用 `i18next` 以支援多國語系
*   **開發工具:** `nodemon` 用於開發時自動重啟伺服器
*   **打包工具:** `esbuild` 用於建立生產環境的打包檔案
